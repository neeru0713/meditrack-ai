# MediTrack AI

Intelligent personal health management platform built with Next.js 15, TypeScript, MongoDB, NextAuth v5, and Vercel AI SDK + Groq.

## Features
- User-linked health record CRUD for vitals, medications, appointments, symptoms.
- Streaming AI endpoints for insights, symptom chat, and weekly digest.
- SSR dashboard with trend charts.
- Middleware route protection.
- Compound MongoDB indexes for time-series access.
- CI workflow running Vitest tests.

## Run locally
1. `cp .env.example .env.local`
2. Fill env values.
3. `npm install`
4. `npm run dev`
